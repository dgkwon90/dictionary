# desktop-ui

Tauri 2 기반 데스크톱 UI. 트레이, 글로벌 단축키, 알림, 화면(Quick Search/Inbox/Result Detail/Review/Dashboard/Settings)을 담당하고, `apps/desktopd`와는 로컬 HTTP API로만 통신한다.

아직 코드 없음. 프레임워크(React/Svelte)는 `../../docs/adr/ADR-0005-frontend-framework.md`에서 확정 후 백로그 `../../docs/planning/backlog.md` #13(셸 부트스트랩)부터 시작한다.
